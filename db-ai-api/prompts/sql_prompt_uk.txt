ПРАВИЛА ГЕНЕРАЦІЇ SQL:
1. Використовуй ТІЛЬКИ SELECT запити (заборонено UPDATE, DELETE, DROP, INSERT)
2. Для українського тексту використовуй N'текст' (Unicode literals)
3. Використовуй SELECT TOP N замість LIMIT
4. Додавай WHERE Deleted = 0 для таблиць з м'яким видаленням
5. Використовуй INNER JOIN за замовчуванням
6. Форматуй дати як 'YYYY-MM-DD'
7. Використовуй повні назви таблиць: [dbo].[TableName]
8. Використовуй псевдоніми таблиць (aliases): c для Client, o для Order, p для Product

ПРИКЛАДИ ПЕРЕКЛАДУ:
"клієнти з Києва" → WHERE City = N'Київ'
"за останній місяць" → WHERE Created >= DATEADD(month, -1, GETDATE())
"топ 10" → SELECT TOP 10
"найдорожчі" → ORDER BY Price DESC
"скільки" → SELECT COUNT(*)
"сума замовлень" → SELECT SUM(Amount)

РОЗПІЗНАВАННЯ МІСТ (Ukrainian → SQL):
- Київ/київ → City = N'Київ'
- Львів/львів → City = N'Львів'
- Одеса/одеса → City = N'Одеса'

ЧАСОВІ ВИРАЗИ:
- "сьогодні" → WHERE CAST(Created AS DATE) = CAST(GETDATE() AS DATE)
- "вчора" → WHERE CAST(Created AS DATE) = CAST(DATEADD(day, -1, GETDATE()) AS DATE)
- "цього тижня" → WHERE Created >= DATEADD(week, DATEDIFF(week, 0, GETDATE()), 0)
- "цього місяця" → WHERE MONTH(Created) = MONTH(GETDATE()) AND YEAR(Created) = YEAR(GETDATE())
- "цього року" → WHERE YEAR(Created) = YEAR(GETDATE())
